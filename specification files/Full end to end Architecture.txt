AI-Powered Forensic Triage & Postmortem Intelligence Platform

PART I — SYSTEM OVERVIEW
1.1 What AIVENTRA Is
AIVENTRA is a multi-agent AI forensic triage platform designed for law enforcement, medical examiner offices, and criminal investigation bureaus. It ingests heterogeneous forensic evidence — autopsy reports, CCTV metadata, mobile device records, location history, financial transactions, social media exports — and runs a coordinated pipeline of specialized AI agents that each process one domain of evidence. The outputs of all agents are fused by a Hypothesis Manager that maintains competing hypotheses (homicide, suicide, accident, natural causes) ranked by posterior probability.

The system is not a replacement for human judgment. It is a triage and intelligence acceleration layer that:

Reduces time-to-first-hypothesis from days to under an hour

Surfaces cross-domain correlations that a human team might take weeks to find manually

Maintains complete cryptographic evidence integrity and chain of custody

Provides explainable, citable conclusions that investigators can verify and contest

The entire architecture is built for hackathon-grade speed (36-hour build cycles) while meeting production-grade security requirements, because evidence handling in law enforcement has zero tolerance for data leakage, tampering, or admission failures.

1.2 Core Design Axioms
Every architectural decision traces back to these five axioms:

Axiom 1 — Evidence Immutability: Once evidence is uploaded, its bytes and their cryptographic fingerprint are permanent. No agent, no investigator, and no admin can silently alter evidence. Tampering is detected and blocked.

Axiom 2 — PII Never Leaves the Boundary: No personally identifiable information — victim names, witness identities, addresses, phone numbers, Aadhaar/SSN — ever reaches an external LLM API. All masking is local, all mapping is ephemeral, all de-anonymization is role-controlled.

Axiom 3 — Zero Trust Between Components: Every call between services is authenticated and authorized. Agents do not trust each other. The Orchestrator does not implicitly trust agents. Every inter-service call carries a cryptographic task token and mTLS certificate.

Axiom 4 — Explainability Is Not Optional: Every hypothesis score, every anomaly flag, every TOD estimate must carry an explanation trace — which evidence contributed, how much, in which direction. This is required for court admissibility and for investigators to contest AI conclusions.

Axiom 5 — Uncertainty Is a First-Class Output: Probability intervals, confidence bands, consistency flags, and "insufficient data" signals are as important as point estimates. A system that hides its uncertainty is more dangerous than a less capable system that is honest about what it does not know.

1.4 Non-Goals
The following are explicitly outside scope for v1:

Real-time audio/video analysis of CCTV footage (metadata only, not frames)

Autonomous agent actions (no agent writes files or sends messages)

Integration with police department databases (CCTNS, AFIS) in v1 — planned for v2

Mobile app (web-responsive only)

Multi-jurisdiction multi-tenancy (single-org deployment in v1)

3.3 Real-Time Agent Progress (WebSocket)
Every agent run emits progress events via WebSocket. The frontend maintains a persistent WS connection per active case:

typescript
// hooks/useAgentProgress.ts
export function useAgentProgress(caseId: string) {
  const { lastMessage } = useWebSocket(`/ws/case/${caseId}`);
  const setAgentStatus = useAgentStore(s => s.setStatus);

  useEffect(() => {
    if (!lastMessage) return;
    const event = JSON.parse(lastMessage.data) as AgentEvent;

    switch (event.type) {
      case 'AGENT_STARTED':
        setAgentStatus(event.agentId, { status: 'RUNNING', startedAt: event.ts });
        break;
      case 'AGENT_PROGRESS':
        setAgentStatus(event.agentId, { progress: event.pct, step: event.step });
        break;
      case 'AGENT_COMPLETE':
        setAgentStatus(event.agentId, { status: 'DONE', output: event.outputRef });
        queryClient.invalidateQueries(['agent-output', event.agentId, caseId]);
        break;
      case 'AGENT_ERROR':
        setAgentStatus(event.agentId, { status: 'ERROR', error: event.message });
        break;
      case 'HYPOTHESIS_UPDATED':
        queryClient.invalidateQueries(['hypothesis', caseId]);
        break;
    }
  }, [lastMessage]);
}
3.4 Evidence Upload Flow (Frontend)
typescript
// components/evidence/EvidenceUploader.tsx
async function uploadEvidence(files: File[], caseId: string) {
  for (const file of files) {
    // 1. Compute client-side SHA-256 for pre-upload integrity check
    const buffer = await file.arrayBuffer();
    const hashBuffer = await crypto.subtle.digest('SHA-256', buffer);
    const clientHash = Array.from(new Uint8Array(hashBuffer))
      .map(b => b.toString(16).padStart(2, '0')).join('');

    // 2. Upload with multipart form
    const formData = new FormData();
    formData.append('file', file);
    formData.append('case_id', caseId);
    formData.append('client_hash', clientHash);

    const response = await api.post('/evidence/upload', formData, {
      headers: { 'Content-Type': 'multipart/form-data' },
      onUploadProgress: (e) => setProgress(file.name, e.loaded / e.total),
    });

    // 3. Verify server echoes back matching hash
    if (response.data.sha256 !== clientHash) {
      throw new Error('Upload integrity check failed — hash mismatch');
    }
  }
}
3.5 Reasoning Replay Viewer
The Reasoning Replay Viewer is a specialized UI component that renders the complete inference chain for any hypothesis conclusion:

typescript
interface ReplayStep {
  stepId: string;
  stepType: 'EVIDENCE_READ' | 'AGENT_INFERENCE' | 'FUSION' | 'HYPOTHESIS_UPDATE';
  agentId: string;
  timestamp: string;
  inputSummary: string;
  outputSummary: string;
  evidenceIds: string[];      // Which evidence pieces were used
  featureContributions: { feature: string; value: number; direction: string }[];
  confidenceBefore: number;
  confidenceAfter: number;
  modelVersion: string;
  signature: string;           // RS256 JWT — verifiable
}
The viewer renders these steps as a scrollable timeline with evidence pop-ups, feature importance bars, and a "verify signature" button that calls POST /replay/verify-step to confirm the step was not tampered with.

app = FastAPI(title="AIVENTRA API", version="1.0.0")

# 3. Audit logging (all requests auto-logged)
app.add_middleware(AuditMiddleware)

app.add_middleware(GZipMiddleware, minimum_size=1024)
PART V — AGENT LAYER
5.1 Agent Taxonomy
AIVENTRA has 11 agents organized into 3 tiers:

text
TIER 1 — INGESTION AGENTS
  (Run first, in parallel, as soon as evidence is uploaded)
  ├── EvidenceParserAgent       → Identifies file type, extracts text/metadata
  ├── OCRAgent                  → Extracts text from scanned PDFs/images
  └── FormatNormalizerAgent     → Converts all evidence to canonical schema

TIER 2 — ANALYSIS AGENTS
  (Run after Tier 1 completes; some run in parallel, some are sequenced)
  ├── AutopsyAgent              → Extracts autopsy findings into structured schema
  ├── TODAgent                  → Time-of-death estimation (physics + ML + Monte Carlo)
  ├── DigitalTimelineAgent      → Constructs chronological event timeline from digital sources
  └── AnomalyDetectorAgent      → Flags behavioral anomalies in timeline sequences

TIER 3 — FUSION AGENTS
  (Run after all Tier 2 agents complete)
  ├── HotspotCorrelationAgent   → Finds overlapping anomaly windows across domains
  ├── HypothesisManagerAgent    → Maintains and updates Bayesian hypothesis posteriors
  ├── ReasoningReplayAgent      → Records and structures complete inference chain
  └── ReportGeneratorAgent      → Compiles final role-filtered narrative report
5.2 Agent Execution Model
Each agent is a standalone FastAPI microservice running in its own process (or container in production). Agents do not share memory. All communication is via HTTP with mTLS and task tokens.

python
# agents/base_agent.py — Base class all agents inherit from

class BaseAgent:
    def __init__(self, agent_id: str, port: int):
        self.agent_id   = agent_id
        self.app        = FastAPI(title=f"AIVENTRA — {agent_id}")
        self.port       = port
        self._register_routes()

    def _register_routes(self):

5.4.2 Physics Core (Henssge Model)
The Henssge nomogram is the internationally accepted standard for rectal-temperature-based PMI estimation:

python
import numpy as np
from scipy.optimize import brentq

HENSSGE_CORRECTION_FACTORS = {
    # (clothing, environment) → correction factor
    ("LIGHT",  "INDOOR"):  1.0,
    ("MEDIUM", "INDOOR"):  0.7,
    ("HEAVY",  "INDOOR"):  0.5,
    ("LIGHT",  "OUTDOOR"): 1.1,
    ("MEDIUM", "OUTDOOR"): 0.8,
    ("HEAVY",  "OUTDOOR"): 0.6,
}

def henssge_cooling_model(
    time_h: float,
    temp_ambient: float,
    weight_kg: float,
    correction_factor: float
) -> float:
    """
    Henssge double-exponential cooling model.
    Returns predicted rectal temperature at time_h hours post mortem.
    Based on: Henssge C (1988) Death time estimation in case work.
    """
    B = 1.11 * (weight_kg ** (-0.25)) * correction_factor
    Q = 1.25 * (weight_kg ** (-0.25)) * correction_factor

    # Double exponential: T_rectal = T_ambient + (37 - T_ambient) * (1.25*e^(-B*t) - 0.25*e^(-Q*t))
    temp = temp_ambient + (37.0 - temp_ambient) * (
        1.25 * np.exp(-B * time_h) - 0.25 * np.exp(-Q * time_h)
    )
    return temp


def henssge_estimate_pmi(
    temp_rectal: float,
    temp_ambient: float,
    weight_kg: float,
    correction_factor: float
) -> dict:
    """
    Numerically invert the Henssge model to estimate PMI from measured temperature.
    Returns mean PMI in hours plus 95% uncertainty interval (Henssge ±3.2h tolerance).
    """
    def objective(t):
        return henssge_cooling_model(t, temp_ambient, weight_kg, correction_factor) - temp_rectal

    # PMI must be in range 0–72h; find root
    try:
        pmi_mean = brentq(objective, 0.01, 72.0, xtol=0.01)
    except ValueError:
        raise ValueError("Henssge model could not converge — temperature outside plausible range")

    # ±3.2h uncertainty from Henssge literature
    uncertainty = 3.2
    return {
        "pmi_mean_hours": round(pmi_mean, 2),
        "pmi_lower_95":   round(max(0, pmi_mean - uncertainty), 2),
        "pmi_upper_95":   round(pmi_mean + uncertainty, 2),
    }

def sign_likelihood(pmi_hours: float, sign_stage: str, table: dict) -> float:
    """Gaussian likelihood of observing this sign stage at a given PMI."""
    if sign_stage not in table:
        return 1.0  # No data → flat likelihood
    params = table[sign_stage]
    lo, hi = params["range"]
    if pmi_hours < lo or pmi_hours > hi:
        return 1e-10  # Outside plausible range → near-zero
    mu, sigma = params["mu"], params["sigma"]
    return np.exp(-0.5 * ((pmi_hours - mu) / sigma) ** 2)
5.4.4 Monte Carlo Posterior Fusion
python
def monte_carlo_tod_posterior(
    physics_pmi: dict,
    rigor: Optional[str],
    livor: Optional[str],
    decomp: Optional[str],
    ml_pmi_mean: Optional[float],
    ml_pmi_std: Optional[float],
    found_dead_dt: datetime,
    last_seen_alive_dt: Optional[datetime],
    n_samples: int = 10_000
) -> dict:
    """
    Monte Carlo approximation of p(TOD | all evidence).

    Prior: uniform between last_seen_alive and found_dead (or 72h window).
    Likelihood: product of physics, signs, and ML components.
    Returns posterior histogram + summary statistics.
    """
    # Prior bounds in PMI hours (relative to found_dead)
    prior_max = min(72.0, (found_dead_dt - last_seen_alive_dt).total_seconds() / 3600
                    if last_seen_alive_dt else 72.0)
    prior_samples = np.random.uniform(0.001, prior_max, n_samples)

    log_weights = np.zeros(n_samples)

    # Physics likelihood (Gaussian centered at physics_pmi_mean)
    if physics_pmi:
        mu_p    = physics_pmi["pmi_mean_hours"]
        sigma_p = (physics_pmi["pmi_upper_95"] - physics_pmi["pmi_lower_95"]) / (2 * 1.96)
        log_weights += -0.5 * ((prior_samples - mu_p) / sigma_p) ** 2

    # Sign likelihoods
    if rigor:
        liks = np.array([sign_likelihood(t, rigor, RIGOR_PMI_RANGES) for t in prior_samples])
        log_weights += np.log(liks + 1e-300)
    if livor:
        liks = np.array([sign_likelihood(t, livor, LIVOR_PMI_RANGES) for t in prior_samples])
        log_weights += np.log(liks + 1e-300)
    if decomp:
        liks = np.array([sign_likelihood(t, decomp, DECOMP_PMI_RANGES) for t in prior_samples])
        log_weights += np.log(liks + 1e-300)

    # ML surrogate likelihood
    if ml_pmi_mean is not None and ml_pmi_std is not None:
        log_weights += -0.5 * ((prior_samples - ml_pmi_mean) / ml_pmi_std) ** 2

    # Normalize weights (importance sampling)
    weights = np.exp(log_weights - np.max(log_weights))
    weights /= weights.sum()

    # Posterior PMI samples (resample)
    posterior_pmi = np.random.choice(prior_samples, size=n_samples, p=weights, replace=True)

    # Convert PMI → TOD timestamps
    posterior_tod = [
        (found_dead_dt - timedelta(hours=float(p))).isoformat()
        for p in posterior_pmi
    ]

    # Summary statistics on PMI
    pmi_mean   = float(np.mean(posterior_pmi))
    pmi_median = float(np.median(posterior_pmi))
    pmi_p025   = float(np.percentile(posterior_pmi, 2.5))
    pmi_p975   = float(np.percentile(posterior_pmi, 97.5))
    pmi_p16    = float(np.percentile(posterior_pmi, 16))
    pmi_p84    = float(np.percentile(posterior_pmi, 84))

    tod_mean = found_dead_dt - timedelta(hours=pmi_mean)

    return {
        "tod_point_estimate":   tod_mean.isoformat(),
        "pmi_mean_hours":       round(pmi_mean, 2),
        "tod_window_95": {
            "start": (found_dead_dt - timedelta(hours=pmi_p975)).isoformat(),
            "end":   (found_dead_dt - timedelta(hours=pmi_p025)).isoformat(),
        },
        "tod_window_68": {
            "start": (found_dead_dt - timedelta(hours=pmi_p84)).isoformat(),
            "end":   (found_dead_dt - timedelta(hours=pmi_p16)).isoformat(),
        },
        "n_samples":  n_samples,
        "ess":        round(1.0 / np.sum(weights ** 2)),  # Effective sample size
    }

async def fuse_timelines(event_lists: list[list[TimelineEvent]]) -> list[TimelineEvent]:
    """
    Merge events from all sources, deduplicate near-duplicate timestamps,
    sort chronologically, and assign sequential event IDs.
    """
    all_events = [e for sublist in event_lists for e in sublist]

    # Sort by timestamp
    all_events.sort(key=lambda e: e.timestamp)

    # Deduplicate: events within 2s of each other from same source = same event
    deduped = []
    for event in all_events:
        if deduped and event.source_type == deduped[-1].source_type:
            gap = abs((parse_dt(event.timestamp) - parse_dt(deduped[-1].timestamp)).total_seconds())
            if gap < 2.0:
                continue
        deduped.append(event)

    # Re-assign sequential IDs
    for i, e in enumerate(deduped):
        e.event_id = f"EVT-{i+1:05d}"

    return deduped
5.7 Anomaly Detector Agent — Deep Specification
The Anomaly Detector uses a hybrid unsupervised approach: autoencoder reconstruction error for sequence anomalies, and Isolation Forest for point anomalies in behavioral features.

5.7.1 Feature Engineering
Each timeline event is converted to a feature vector:

python
def event_to_features(event: TimelineEvent, prev_event: Optional[TimelineEvent]) -> np.ndarray:
    """
    Encode a timeline event as a fixed-length numerical feature vector.
    """
    features = []

    # Temporal features
    ts   = parse_dt(event.timestamp)
    hour = ts.hour                                    # 0–23
    dow  = ts.weekday()                               # 0–6
    features += [hour / 23.0, dow / 6.0]

    # Inter-event gap (normalized to 0–1 over 24h)
    if prev_event:
        gap_s = (ts - parse_dt(prev_event.timestamp)).total_seconds()
        features.append(min(gap_s / 86400.0, 1.0))
    else:
        features.append(0.5)

    # Event type encoding (one-hot over known types)
    EVENT_TYPES = ["CALL", "SMS", "LOCATION_PING", "PURCHASE", "APP_USE", "EMAIL", "OTHER"]
    for et in EVENT_TYPES:
        features.append(1.0 if event.event_type == et else 0.0)

    # Location features (distance from home base if known, else 0)
    if event.location:
        features.append(event.location.lat / 90.0)
        features.append(event.location.lng / 180.0)
    else:
        features += [0.0, 0.0]

    return np.array(features, dtype=np.float32)  # 12-dimensional
5.7.2 Isolation Forest (Point Anomalies)

from sklearn.ensemble import IsolationForest

def detect_point_anomalies(
    feature_matrix: np.ndarray,  # Shape: (n_events, n_features)
    contamination: float = 0.05   # Expect ~5% anomalous events
) -> np.ndarray:
    """
    Returns anomaly score per event: negative = more anomalous.
    Normalized to [0, 1] where 1 = most anomalous.
    """
    ifo = IsolationForest(
        n_estimators=200,
        contamination=contamination,
        random_state=42
    )
    ifo.fit(feature_matrix)
    raw_scores = ifo.decision_function(feature_matrix)  # More negative = more anomalous

    # Normalize to [0, 1] (1 = most anomalous)
    norm_scores = (raw_scores.max() - raw_scores) / (raw_scores.max() - raw_scores.min() + 1e-10)
    return norm_scores
5.7.3 Autoencoder (Sequence Anomalies)

import torch
import torch.nn as nn

class TimelineAutoencoder(nn.Module):
    """
    LSTM Autoencoder for temporal sequence reconstruction.
    High reconstruction error = anomalous subsequence.
    """
    def __init__(self, input_dim=12, hidden_dim=32, latent_dim=8, seq_len=20):
        super().__init__()
        self.encoder = nn.LSTM(input_dim, hidden_dim, batch_first=True)
        self.to_latent = nn.Linear(hidden_dim, latent_dim)
        self.from_latent = nn.Linear(latent_dim, hidden_dim)
        self.decoder = nn.LSTM(hidden_dim, input_dim, batch_first=True)

    def forward(self, x):
        # x: (batch, seq_len, input_dim)
        enc_out, (h, c) = self.encoder(x)
        latent = self.to_latent(h[-1])                    # Bottleneck
        decoded_h = self.from_latent(latent).unsqueeze(1).repeat(1, x.size(1), 1)
        dec_out, _ = self.decoder(decoded_h)
        return dec_out, latent

def compute_reconstruction_errors(
    model: TimelineAutoencoder,
    sequences: torch.Tensor
) -> np.ndarray:
    model.eval()
    with torch.no_grad():
        reconstructed, _ = model(sequences)
        errors = ((sequences - reconstructed) ** 2).mean(dim=[1, 2])  # Per-sequence MSE
    return errors.numpy()

5.8 Hotspot Correlation Agent
The Hotspot Agent finds temporal windows where multiple anomaly signals overlap — these windows are the highest-priority intervals for investigator attention.

5.8.1 Algorithm
python
def find_hotspot_windows(
    anomaly_events: list[dict],     # Events with anomaly_score and timestamp
    tod_window: dict,               # TOD 95% credible interval
    min_signals: int = 2,           # Minimum co-occurring anomalies to call a hotspot
    window_hours: float = 2.0       # Sliding window size
) -> list[HotspotWindow]:
    """
    Slide a time window over the full event range.
    Count anomaly signals (score > threshold) in each window.
    Windows with >= min_signals are hotspots.
    Intersect with TOD window to find highest-priority intervals.
    """
    if not anomaly_events:
        return []

    events_sorted = sorted(anomaly_events, key=lambda e: e["timestamp"])
    start = parse_dt(events_sorted[0]["timestamp"])
    end   = parse_dt(events_sorted[-1]["timestamp"])

    hotspots = []
    cursor = start

    while cursor < end:
        window_end = cursor + timedelta(hours=window_hours)

        # Collect anomalous events in this window
        window_events = [
            e for e in events_sorted
            if cursor <= parse_dt(e["timestamp"]) < window_end
            and e.get("anomaly_score", 0) > 0.6
        ]

        # Count unique anomaly types
        anomaly_types = set(
            flag for e in window_events for flag in e.get("anomaly_flags", [])
        )

        if len(anomaly_types) >= min_signals:
            # Check overlap with TOD window
            tod_overlap = overlaps_tod_window(cursor, window_end, tod_window)

            hotspots.append(HotspotWindow(
                start=cursor.isoformat(),
                end=window_end.isoformat(),
                anomaly_count=len(window_events),
                anomaly_types=list(anomaly_types),
                composite_score=sum(e["anomaly_score"] for e in window_events) / len(window_events),
                tod_window_overlap=tod_overlap,
                priority="CRITICAL" if tod_overlap and len(anomaly_types) >= 3 else
                          "HIGH"     if tod_overlap or len(anomaly_types) >= 3 else
                          "MEDIUM"
            ))

        cursor += timedelta(minutes=30)   # 30-minute slide step

    # Merge overlapping hotspots
    return merge_overlapping_hotspots(hotspots)

5.9.2 Bayesian Update Algorithm
python
HYPOTHESES = ["HOMICIDE", "SUICIDE", "ACCIDENT", "NATURAL", "UNDETERMINED"]

class HypothesisManager:
    def __init__(self):
        # Equal priors initially
        self.posteriors = {h: 1.0 / len(HYPOTHESES) for h in HYPOTHESES}
        self.evidence_trail: list[dict] = []

    def update(self, signal_key: str, confidence: float = 1.0):
        """
        Apply one piece of evidence to the posterior.
        Confidence scales the likelihood multipliers (0 = no update, 1 = full update).
        Records the update in the evidence trail for Reasoning Replay.
        """
        if signal_key not in EVIDENCE_LIKELIHOOD_TABLE:
            return  # Unknown signal — no update

        likelihoods = EVIDENCE_LIKELIHOOD_TABLE[signal_key]
        before = dict(self.posteriors)

        # Bayesian update: posterior ∝ prior × likelihood
        for h in HYPOTHESES:
            if h == "UNDETERMINED":
                continue  # UNDETERMINED is residual
            multiplier = likelihoods.get(h, 1.0)
            # Scale toward 1.0 proportional to (1 - confidence) for uncertain signals
            effective_multiplier = 1.0 + (multiplier - 1.0) * confidence
            self.posteriors[h] *= effective_multiplier

        # Normalize (excluding UNDETERMINED for now)
        total = sum(v for k, v in self.posteriors.items() if k != "UNDETERMINED")
        for h in HYPOTHESES:
            if h != "UNDETERMINED":
                self.posteriors[h] /= total

        # UNDETERMINED = entropy of distribution (high uncertainty = high UNDETERMINED)
        max_p    = max(self.posteriors[h] for h in HYPOTHESES if h != "UNDETERMINED")
        self.posteriors["UNDETERMINED"] = max(0.0, 1.0 - max_p * 2.5)

        # Renormalize including UNDETERMINED
        total = sum(self.posteriors.values())
        for h in HYPOTHESES:
            self.posteriors[h] /= total

        # Record for replay
        self.evidence_trail.append({
            "signal":  signal_key,
            "confidence": confidence,
            "before":  before,
            "after":   dict(self.posteriors),
            "delta":   {h: self.posteriors[h] - before[h] for h in HYPOTHESES},
        })

    def top_hypothesis(self) -> tuple[str, float]:
        best = max(HYPOTHESES, key=lambda h: self.posteriors[h])
        return best, self.posteriors[best]

@router.post("/replay/step/{step_id}/verify")
async def verify_replay_step(
    step_id: str,
    user = Depends(require_role("admin", "lead_investigator", "investigator"))
):
    """
    Cryptographically verify a replay step's signature.
    Used by the frontend 'Verify' button in the Replay Viewer.
    """
    step = await db.fetchone("SELECT * FROM replay_steps WHERE step_id = $1", step_id)
    if not step:
        raise HTTPException(404, "Step not found")

    # Verify RS256 signature
    try:
        claims = jwt.decode(step["signature"], ORCH_PUBLIC_KEY, algorithms=["RS256"])
    except jwt.InvalidTokenError as e:
        return {"valid": False, "reason": str(e)}

    # Recompute step hash
    step_dict = dict(step)
    for field in ["step_hash", "signature"]:
        step_dict.pop(field, None)
    canonical = json.dumps(step_dict, sort_keys=True, separators=(",", ":"),
                           default=str)
    computed_hash = hashlib.sha256(canonical.encode()).hexdigest()

    if computed_hash != claims.get("step_hash"):
        return {"valid": False, "reason": "Step content hash mismatch — tampered"}

    return {
        "valid":      True,
        "agent_id":   claims["agent_id"],
        "case_id":    claims["case_id"],
        "signed_at":  claims["signed_at"],
        "step_id":    step_id,
    }

PART VII — ORCHESTRATION ENGINE
7.1 Orchestrator Architecture
The Orchestrator is the central scheduler. It reads from a Redis Streams task queue, resolves the agent DAG, dispatches agents via mTLS HTTP, monitors progress via polling, handles retries, and emits WebSocket events.

python
# orchestrator/orchestrator.py

import asyncio
import httpx
from datetime import datetime

class AgentOrchestrator:
    """
    Central orchestration engine for AIVENTRA.

    Lifecycle per case run:
      1. Receive trigger (from Core API via Redis Streams)
      2. Create AgentRun records for all 11 agents
      3. Compute initial ready set (Tier 1 agents — no deps)
      4. Dispatch ready agents in parallel
      5. For each completed agent:
           a. Mark AgentRun as DONE in DB
           b. Re-compute ready set
           c. Dispatch newly unblocked agents
      6. When all agents done: emit PIPELINE_COMPLETE event
    """

    def __init__(self):
        self._mTLS_client = httpx.AsyncClient(
            cert=("certs/orchestrator.crt", "keys/orchestrator.key"),
            verify="certs/internal_ca.crt",
            timeout=httpx.Timeout(connect=5.0, read=300.0, write=30.0)
        )
        self._ws_manager  = ws_manager
        self._circuit_breakers = {agent: CircuitBreaker(agent) for agent in AGENT_DAG}

    async def run_pipeline(self, case_id: str, triggered_by: str, config: dict):
        """Entry point: kick off the full agent pipeline for a case."""
        pipeline_id = str(uuid.uuid4())

        # Create AgentRun rows for all agents
        async with get_db() as db:
            for agent_id in AGENT_DAG:
                await db.execute("""
                    INSERT INTO agent_runs
                        (case_id, agent_id, triggered_by, status, tier, config_snapshot)
                    VALUES ($1, $2, $3, 'PENDING', $4, $5)
                """, case_id, agent_id, triggered_by,
                    AGENT_DAG[agent_id]["tier"],
                    json.dumps(config))

        await self._ws_manager.broadcast(case_id, {
            "type":        "PIPELINE_STARTED",
            "pipeline_id": pipeline_id,
            "case_id":     case_id,
            "total_agents": len(AGENT_DAG),
            "ts":          datetime.utcnow().isoformat()
        })

        await log_audit_event("PIPELINE_STARTED", "SUCCESS",
                               case_id=case_id, context={"pipeline_id": pipeline_id})

        # Dispatch Tier 1 immediately
        ready = await self._get_ready_agents(case_id)
        await asyncio.gather(*[self._dispatch_agent(agent_id, case_id, config)
                                for agent_id in ready])

    async def _dispatch_agent(self, agent_id: str, case_id: str, config: dict):
        """Dispatch a single agent and handle its completion/failure."""
        cb = self._circuit_breakers[agent_id]
        if cb.is_open:
            await self._handle_agent_failure(
                agent_id, case_id,
                f"Circuit breaker OPEN for {agent_id} — skipping"
            )
            return

        # Issue task token (Layer 7)
        task_token = issue_task_token(
            target=agent_id,
            case_id=case_id,
            operation="run",
            ttl=300
        )

        # Mark as RUNNING
        async with get_db() as db:
            await db.execute("""
                UPDATE agent_runs SET status = 'RUNNING', started_at = NOW()
                WHERE case_id = $1 AND agent_id = $2
            """, case_id, agent_id)

        await self._ws_manager.broadcast(case_id, {
            "type":     "AGENT_STARTED",
            "agent_id": agent_id,
            "ts":       datetime.utcnow().isoformat()
        })

        try:
            agent_port = AGENT_PORTS[agent_id]
            response   = await self._mTLS_client.post(
                f"https://{agent_id}:{agent_port}/run",
                json={
                    "case_id":    case_id,
                    "task_token": task_token,
                    "config":     config,
                },
                timeout=httpx.Timeout(connect=5.0, read=300.0, write=30.0)
            )
            response.raise_for_status()
            output = response.json()

            # Mark complete
            async with get_db() as db:
                await db.execute("""
                    UPDATE agent_runs
                    SET status = 'DONE', completed_at = NOW(),
                        duration_ms = EXTRACT(EPOCH FROM (NOW() - started_at)) * 1000,
                        llm_tokens_used = $3
                    WHERE case_id = $1 AND agent_id = $2
                """, case_id, agent_id, output.get("llm_tokens_used", 0))

            cb.record_success()

            await self._ws_manager.broadcast(case_id, {
                "type":     "AGENT_COMPLETE",
                "agent_id": agent_id,
                "ts":       datetime.utcnow().isoformat()
            })

            # Dispatch newly unblocked agents
            next_ready = await self._get_ready_agents(case_id)
            if next_ready:
                await asyncio.gather(*[
                    self._dispatch_agent(nid, case_id, config) for nid in next_ready
                ])
            else:
                # Check if pipeline is fully complete
                await self._check_pipeline_complete(case_id)

        except Exception as e:
            cb.record_failure()
            await self._handle_agent_failure(agent_id, case_id, str(e))

    async def _handle_agent_failure(self, agent_id: str, case_id: str, error: str):
        async with get_db() as db:
            run = await db.fetchrow(
                "SELECT retry_count FROM agent_runs WHERE case_id = $1 AND agent_id = $2",
                case_id, agent_id
            )
            retry_count = run["retry_count"] if run else 0

            if retry_count < MAX_RETRIES:
                # Exponential backoff retry
                delay = (2 ** retry_count) * 5
                await asyncio.sleep(delay)
                await db.execute("""
                    UPDATE agent_runs SET status = 'PENDING', retry_count = retry_count + 1
                    WHERE case_id = $1 AND agent_id = $2
                """, case_id, agent_id)
                await self._dispatch_agent(agent_id, case_id, {})
            else:
                await db.execute("""
                    UPDATE agent_runs
                    SET status = 'FAILED', completed_at = NOW(), error_message = $3
                    WHERE case_id = $1 AND agent_id = $2
                """, case_id, agent_id, error)
                await self._ws_manager.broadcast(case_id, {
                    "type":     "AGENT_ERROR",
                    "agent_id": agent_id,
                    "message":  error,
                    "ts":       datetime.utcnow().isoformat()
                })
                await log_audit_event("AGENT_RUN_FAILURE", "ERROR",
                                       agent_id=agent_id, case_id=case_id,
                                       context={"error": error})

    async def _get_ready_agents(self, case_id: str) -> list[str]:
        async with get_db() as db:
            rows = await db.fetch(
                "SELECT agent_id, status FROM agent_runs WHERE case_id = $1", case_id
            )
        done    = {r["agent_id"] for r in rows if r["status"] == "DONE"}
        running = {r["agent_id"] for r in rows if r["status"] == "RUNNING"}
        return [
            aid for aid, cfg in AGENT_DAG.items()
            if aid not in done and aid not in running
            and all(dep in done for dep in cfg["depends_on"])
        ]

    async def _check_pipeline_complete(self, case_id: str):
        async with get_db() as db:
            rows = await db.fetch(
                "SELECT status FROM agent_runs WHERE case_id = $1", case_id
            )
        statuses = {r["status"] for r in rows}
        if statuses <= {"DONE", "FAILED"}:
            failed = [r for r in rows if r["status"] == "FAILED"]
            await self._ws_manager.broadcast(case_id, {
                "type":         "PIPELINE_COMPLETE",
                "case_id":      case_id,
                "failed_agents": [r["agent_id"] for r in failed] if failed else [],
                "ts":           datetime.utcnow().isoformat()
            })
7.2 Circuit Breaker
python
# orchestrator/circuit_breaker.py

from enum import Enum
import time

class CBState(Enum):
    CLOSED   = "CLOSED"    # Normal — requests pass through
    OPEN     = "OPEN"      # Failing — requests blocked
    HALF_OPEN = "HALF_OPEN" # Testing — one probe request allowed

class CircuitBreaker:
    """
    Per-agent circuit breaker.
    Prevents cascade failures from a broken agent repeatedly consuming retries.
    """
    FAILURE_THRESHOLD = 3     # Open after 3 consecutive failures
    RECOVERY_TIMEOUT  = 60    # Seconds before moving to HALF_OPEN
    SUCCESS_THRESHOLD = 1     # Successes needed to close from HALF_OPEN

    def __init__(self, agent_id: str):
        self.agent_id         = agent_id
        self.state            = CBState.CLOSED
        self.failure_count    = 0
        self.last_failure_ts  = None
        self.success_count    = 0

    @property
    def is_open(self) -> bool:
        if self.state == CBState.OPEN:
            if time.time() - self.last_failure_ts > self.RECOVERY_TIMEOUT:
                self.state         = CBState.HALF_OPEN
                self.success_count = 0
                return False
            return True
        return False

    def record_success(self):
        if self.state == CBState.HALF_OPEN:
            self.success_count += 1
            if self.success_count >= self.SUCCESS_THRESHOLD:
                self.state         = CBState.CLOSED
                self.failure_count = 0
        elif self.state == CBState.CLOSED:
            self.failure_count = 0

    def record_failure(self):
        self.failure_count   += 1
        self.last_failure_ts  = time.time()
        if self.failure_count >= self.FAILURE_THRESHOLD:
            self.state = CBState.OPEN

# orchestrator/task_queue.py

import asyncio
import aioredis

STREAM_KEY    = "aiventra:pipeline:tasks"
CONSUMER_GROUP = "orchestrators"
CONSUMER_NAME  = "orchestrator-1"

async def setup_stream(redis: aioredis.Redis):
    """Create consumer group if not exists."""
    try:
        await redis.xgroup_create(STREAM_KEY, CONSUMER_GROUP, id="0", mkstream=True)
    except aioredis.ResponseError as e:
        if "BUSYGROUP" not in str(e):
            raise

async def enqueue_pipeline(redis: aioredis.Redis, case_id: str,
                             triggered_by: str, config: dict):
    """Enqueue a pipeline run request."""
    await redis.xadd(STREAM_KEY, {
        "case_id":      case_id,
        "triggered_by": triggered_by,
        "config":       json.dumps(config),
        "enqueued_at":  datetime.utcnow().isoformat()
    })

async def consume_tasks(orchestrator: AgentOrchestrator, redis: aioredis.Redis):
    """Blocking consumer loop — processes pipeline tasks from Redis Stream."""
    await setup_stream(redis)

    while True:
        try:
            messages = await redis.xreadgroup(
                CONSUMER_GROUP, CONSUMER_NAME,
                {STREAM_KEY: ">"},
                count=1,
                block=1000   # Block up to 1s waiting for new messages
            )

            for _, entries in (messages or []):
                for msg_id, fields in entries:
                    try:
                        await orchestrator.run_pipeline(
                            case_id=fields["case_id"],
                            triggered_by=fields["triggered_by"],
                            config=json.loads(fields.get("config", "{}"))
                        )
                        # Acknowledge message
                        await redis.xack(STREAM_KEY, CONSUMER_GROUP, msg_id)
                    except Exception as e:
                        await log_audit_event("PIPELINE_QUEUE_ERROR", "ERROR",
                                               context={"msg_id": msg_id, "error": str(e)})
                        # Don't ack — message will be retried by another consumer

        except asyncio.CancelledError:
            break
        except Exception as e:
            await asyncio.sleep(5)  # Back off on connection errors

# services/websocket_manager.py

from fastapi import WebSocket
import asyncio
from collections import defaultdict

class WebSocketManager:
    """
    Manages WebSocket connections per case.
    Supports fan-out broadcast (one event → all connected clients for a case).
    """
    def __init__(self):
        # case_id → set of active WebSocket connections
        self._connections: dict[str, set[WebSocket]] = defaultdict(set)
        self._lock = asyncio.Lock()

    async def connect(self, ws: WebSocket, case_id: str):
        await ws.accept()
        async with self._lock:
            self._connections[case_id].add(ws)

    async def disconnect(self, ws: WebSocket, case_id: str):
        async with self._lock:
            self._connections[case_id].discard(ws)
            if not self._connections[case_id]:
                del self._connections[case_id]

    async def broadcast(self, case_id: str, message: dict):
        """Send a message to all WebSocket clients subscribed to a case."""
        payload = json.dumps(message)
        dead    = set()

        async with self._lock:
            connections = set(self._connections.get(case_id, set()))

        for ws in connections:
            try:
                await ws.send_text(payload)
            except Exception:
                dead.add(ws)

        # Clean up dead connections
        if dead:
            async with self._lock:
                self._connections[case_id] -= dead

ws_manager = WebSocketManager()

# WebSocket route
@router.websocket("/ws/case/{case_id}")
async def case_websocket(ws: WebSocket, case_id: str):
    # Authenticate via token cookie on handshake
    token_cookie = ws.cookies.get("access_token")
    if not token_cookie:
        await ws.close(code=4001, reason="Unauthenticated")
        return
    try:
        user = jwt.decode(token_cookie, AUTH_PUBLIC_KEY, algorithms=["RS256"])
    except jwt.InvalidTokenError:
        await ws.close(code=4001, reason="Invalid token")
        return

    # RBAC: check case assignment
    if not await is_case_accessible(case_id, user["sub"], user["role"]):
        await ws.close(code=4003, reason="Forbidden")
        return

    await ws_manager.connect(ws, case_id)
    try:
        # Send current state on connect
        current_state = await build_case_state_summary(case_id)
        await ws.send_text(json.dumps({"type": "INITIAL_STATE", "data": current_state}))

        # Keep alive
        while True:
            try:
                await asyncio.wait_for(ws.receive_text(), timeout=30.0)
            except asyncio.TimeoutError:
                await ws.send_text(json.dumps({"type": "PING"}))
    except Exception:
        pass
    finally:
        await ws_manager.disconnect(ws, case_id)

#!/bin/bash
# scripts/gen_certs.sh — Run once at deployment setup

set -euo pipefail
mkdir -p certs keys

# ── 1. Internal CA ─────────────────────────────────────────────────────────
openssl genrsa -out keys/internal_ca.key 4096
openssl req -new -x509 -key keys/internal_ca.key \
    -out certs/internal_ca.crt -days 3650 \
    -subj "/C=IN/O=AIVENTRA/CN=AIVENTRA Internal CA"

# ── 2. Orchestrator cert ────────────────────────────────────────────────────
openssl genrsa -out keys/orchestrator.key 2048
openssl req -new -key keys/orchestrator.key \
    -out /tmp/orchestrator.csr \
    -subj "/C=IN/O=AIVENTRA/CN=orchestrator"
openssl x509 -req -in /tmp/orchestrator.csr \
    -CA certs/internal_ca.crt -CAkey keys/internal_ca.key \
    -CAcreateserial -out certs/orchestrator.crt -days 365

# ── 3. Agent certs (one per agent) ─────────────────────────────────────────
AGENTS="evidence_parser ocr format_normalizer autopsy tod digital_timeline
        anomaly_detector hotspot_correlation hypothesis_manager
        reasoning_replay report_generator"

for agent in $AGENTS; do
    openssl genrsa -out keys/${agent}.key 2048
    openssl req -new -key keys/${agent}.key \
        -out /tmp/${agent}.csr \
        -subj "/C=IN/O=AIVENTRA/CN=${agent}"
    openssl x509 -req -in /tmp/${agent}.csr \
        -CA certs/internal_ca.crt -CAkey keys/internal_ca.key \
        -CAcreateserial -out certs/${agent}.crt -days 365
done

# ── 4. JWT auth keys (RS256) ────────────────────────────────────────────────
openssl genrsa -out keys/auth_private.pem 2048
openssl rsa -in keys/auth_private.pem -pubout -out keys/auth_public.pem

# ── 5. Output signing keys (RS256) ─────────────────────────────────────────
openssl genrsa -out keys/orchestrator_private.pem 2048
openssl rsa -in keys/orchestrator_private.pem -pubout \
    -out keys/orchestrator_public.pem

echo "All certificates and keys generated."
echo "IMPORTANT: Secure all .key files immediately. Never commit to Git."

@app.get("/ready")
async def ready():
    """
    Readiness probe — returns 200 only if all dependencies are reachable.
    Nginx only routes traffic once this passes.
    """
    checks = {}

    # DB check
    try:
        async with get_db() as db:
            await db.fetchval("SELECT 1")
        checks["database"] = "ok"
    except Exception as e:
        checks["database"] = f"FAIL: {e}"

    # Redis check
    try:
        await redis_client.ping()
        checks["redis"] = "ok"
    except Exception as e:
        checks["redis"] = f"FAIL: {e}"

    # LLM gateway check (for agents that use LLM)
    if hasattr(settings, "GEMINI_API_KEY"):
        checks["llm_gateway"] = "ok"  # Assume ok — actual check is expensive

    all_ok = all(v == "ok" for v in checks.values())
    return JSONResponse(
        status_code=200 if all_ok else 503,
        content={"ready": all_ok, "checks": checks}
    )

# config/logging.py
import logging
import json
from datetime import datetime

class JSONFormatter(logging.Formatter):
    def format(self, record: logging.LogRecord) -> str:
        log_entry = {
            "ts":      datetime.utcnow().isoformat(),
            "level":   record.levelname,
            "service": getattr(record, "service", "unknown"),
            "msg":     record.getMessage(),
            "module":  record.module,
            "func":    record.funcName,
            "line":    record.lineno,
        }
        if record.exc_info:
            log_entry["exception"] = self.formatException(record.exc_info)
        if hasattr(record, "request_id"):
            log_entry["request_id"] = record.request_id
        if hasattr(record, "case_id"):
            log_entry["case_id"] = record.case_id
        return json.dumps(log_entry)

def setup_logging(service_name: str):
    handler = logging.StreamHandler()
    handler.setFormatter(JSONFormatter())
    logger = logging.getLogger()
    logger.addHandler(handler)
    logger.setLevel(logging.INFO)
    logging.LoggerAdapter(logger, {"service": service_name})

13.1 Nine-Layer Defense Matrix

L1	Identity & Authentication	Credential theft, session hijack, replay attacks	bcrypt(14), RS256 JWT, TOTP, Redis blacklist, httpOnly cookies

L6	Audit Log	Insider threats, unlogged access, legal non-compliance	Append-only chained hash log, anomaly detection, all HTTP requests logged
L7	mTLS Call Graph Enforcement	Rogue agents, inter-service impersonation	Internal PKI, client certs on all agent calls, task tokens per dispatch
L8	Prompt Injection Defense	Evidence-embedded LLM hijacking, output manipulation	12 injection pattern detectors, delimiters, Pydantic output validation
L9	AI Gateway + Output Signing	Unauthorized LLM usage, tampered conclusions	Centralized LLM proxy, token rate limits, PII residual scan, RS256 signing

APPENDIX B — GLOSSARY
Term	Definition
PMI	Postmortem Interval — time elapsed since death
TOD	Time of Death — the actual timestamp of death
Henssge Model	Rectal temperature-based PMI estimation model (1988)
PII	Personally Identifiable Information
mTLS	Mutual TLS — both client and server present certificates
Task Token	Short-lived JWT authorizing exactly one agent call
Replay Step	A signed record of one inference decision in the pipeline
Hotspot	A temporal window where multiple anomaly signals co-occur
Hypothesis Posterior	Bayesian probability distribution over {HOMICIDE, SUICIDE, ACCIDENT, NATURAL, UNDETERMINED}
Chained Hash Log	An append-only log where each entry hashes the previous, making deletion detectable
Output Signing	RS256 JWT attached to every agent output, enabling tamper detection
PII Vault	Encrypted, case-scoped mapping between PII surface forms and placeholder tokens
Circuit Breaker	Per-agent fault isolation mechanism that stops dispatching to a repeatedly failing agent
Correction Factor	Henssge model parameter encoding clothing and environment effects on body cooling rate
Isolation Forest	Unsupervised ML anomaly detector that isolates anomalies via random partitioning
LSTM Autoencoder	Neural network that learns normal temporal sequences; high reconstruction error = anomaly
Canonical JSON	JSON serialization with sorted keys and no whitespace — deterministic for hashing
Evidence Quarantine	Soft-delete mechanism that flags suspicious files without destroying them
Role Gate	Frontend component that renders children only for specified roles — UX only, not security
